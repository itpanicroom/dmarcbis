# dmarcbis
DMARCbis builder, humanizer and Azure deployment toolkit.

dmarcbis helps Microsoft 365 administrators and consultants:

Build DMARC records using guided configuration
Deploy DMARC records to Azure DNS using Infrastructure as Code
Parse DMARC aggregate reports
Convert XML reports into human-readable findings
Visualize DMARC trends over time
Generate actionable recommendations for email authentication posture
Vision

Most DMARC tools focus on record validation.

dmarcbis focuses on understanding.

The goal is to help administrators understanding what is happening when a DMARC report kicks in.
